# Tepkiler (Reactions)

Bu klasöre MP3 formatında tepki sesleri ekleyebilirsiniz.

## Dosya Adlandırma

Dosya isimleri otomatik olarak düzenlenir:
- Dosya adı: `ya-sabir.mp3`
- Gösterilecek isim: **Ya Sabir**

### Kurallar:
- Tire (`-`) ile ayrılmış kelimeler
- Her kelimenin ilk harfi büyük yapılır
- `.mp3` uzantısı kaldırılır

## Örnekler:

| Dosya Adı | Gösterilecek İsim |
|-----------|-------------------|
| `ya-sabir.mp3` | Ya Sabir |
| `vine-boom.mp3` | Vine Boom |
| `ulan-sende-bi-sg-amk.mp3` | Ulan Sende Bi Sg Amk |

## Kullanım

1. MP3 dosyasını bu klasöre ekleyin
2. Sunucuyu yeniden başlatın (veya sayfa yenilendiğinde otomatik yüklenir)
3. "Tepkiler" sekmesinde görünecektir

## Öneriler

- Dosya boyutu: < 500KB
- Süre: 1-5 saniye arası
- Format: MP3
- Kalite: 128kbps yeterli
