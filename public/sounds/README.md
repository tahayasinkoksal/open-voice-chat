# Ses Dosyaları

Bu klasöre aşağıdaki ses dosyalarını koyun:

- `join.mp3` - Kullanıcı odaya katıldığında çalacak ses
- `disconnect.mp3` - Kullanıcı odadan ayrıldığında çalacak ses

## Dosya Formatı
- Format: MP3
- Önerilen süre: 1-2 saniye
- Önerilen boyut: < 50KB

## Kullanım
Ses dosyaları `/sounds/` URL'i üzerinden erişilebilir.
